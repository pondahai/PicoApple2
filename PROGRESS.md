# 🍎 PicoApple2 - 專案進度表

## 🚀 當前版本：v1.0-RC (DMA Optimized)
專案已成功遷移至 Rust 核心與 RP2040，並全面實施了 **TFT DMA 掃描線雙緩衝** 渲染，實現了高效能、全速模擬與 SD 卡持久化管理。

---

## ✅ 已完成功能
*   **🦀 Rust 模擬核心**: 完成 6502 模擬、記憶體映射與 Disk II 控制器。
*   **📺 繪圖系統**:
    *   **TFT DMA 渲染**: 實施 Hardware SPI + DMA 傳輸，徹底解決渲染阻塞 CPU 問題。
    *   **Text/Hires**: 支援雙倍寬度模式、彩色與黑白切換、模擬 NTSC 混色。
    *   **Lo-Res**: 補全低解析度繪圖模式 (LGR 40x48) 與 Mixed Mode 渲染。
*   **⚡ 效能優化**: 
    *   RP2040 穩定超頻至 **250MHz**。
    *   **[最新] 啟動同步**: 同步雙核啟動時序，消除 Beep 聲與畫面的時間差。
*   **💾 磁碟系統**: 
    *   支援 `.DSK` 檔案掛載與切換。
    *   **[最新] 磁碟持久化**: 支援開機自動載入 `/LASTDISK.TXT` 記錄的上次磁碟。
    *   實施 **Safe Write-back**：物理驗證磁軌寫回。
    *   **[最新] 週期級同步修復**: 實施了位元組級同步脈衝與寫入模式。
    *   **[最新] 位元級移位暫存器**: 完成了 4-cycle granularity 的 bit-shift 模擬，精確還原 Disk II 硬體行為。

## ⚠️ 開發瓶頸 (Blockers)
*   **🛠️ 磁碟寫入不穩定 (ERROR #8)**: 
    *   位元級邏輯已實施，仍需驗證是否徹底解決 `INIT HELLO` 的寫入錯誤。

---

## 🎮 實體控制系統
*   **完整引腳對應**: 補全所有實體按鈕映射（UP/DOWN/LEFT/RIGHT/A/B/MENU/ALT）。
*   **智慧導航**: 選單模式下自動切換方向鍵邏輯，支援限速滾動。
*   **虛擬搖桿**: 鍵盤實體映射鍵完整模擬 Apple II 雙軸搖桿。

## 🌐 專業虛擬控制台 (Pro Console)
*   **WebSerial 介面**: 開發基於 Chrome/Edge 的網頁控制台，取代傳統 ANSI 終端機。
*   **ANSI 解析修復**: 完整支援終端機 F1-F3 系統功能鍵。
*   **即時狀態協定 (STX)**: 實施 Press/Release 雙態同步，解決雲端/遠端按鍵重複問題。
