# PicoApple2 - 專案進度報告

## 🚀 目前狀態：1.0-RC (Performance Optimized)
本專案已成功移植 Rust 核心至 RP2040，實現了全速 1.023MHz 模擬，並支援磁碟讀寫與 SD 卡檔案管理。

---

## ✅ 已完成功能
*   **🦀 Rust 核心集成**: 完善的 6502 模擬、記憶體映射與 Disk II 控制。
*   **🎨 繪圖系統**:
    *   **Text/Hires**: 支援文字與高解析度模式，具備正確的閃爍、反白與 NTSC 偽色模擬。
    *   **Lo-Res**: 補全低解析度繪圖模式 (LGR 40x48) 與 Mixed Mode 渲染。
*   **⚡ 效能優化**: 
    *   RP2040 穩定超頻至 **250MHz**。
    *   實作 **Disk Boost**：磁碟馬達運轉時自動放開 CPU 限速。
*   **💾 磁碟系統**: 
    *   支援 `.DSK` 檔案選擇與載入。
    *   實作 **Safe Write-back**：物理驗證磁軌寫入。
    *   **持久化狀態**: 新增 `/LASTDISK.TXT` 記憶功能，開機會自動載入上一次使用的磁碟。
*   **🎮 實體控制系統**:
    *   **完整引腳對齊**: 補全所有實體按鈕映射（UP/DOWN/LEFT/RIGHT/A/B/MENU/ALT）。
    *   **MENU 鍵 (START)**: 實作 GPIO 4 (START) 直接呼叫磁碟選單功能。
    *   **虛擬搖桿**: 四向實體按鍵完美模擬 Apple II 雙軸搖桿。
*   **🛠️ 開發工具鏈**:
    *   **環境自動掃描**: 實作 `scan_env.ps1` 與 `check_env.bat`，自動從 Arduino IDE 設定抓取路徑，達成零硬編碼配置。
    *   **Git 管理**: 修正排除規則，將自動產生的 `build_env.bat` 排除於版本控制之外，保護本機開發路徑。
*   **🖥️ 專業虛擬控制台 (Pro Console)**:
    *   **WebSerial 整合**: 開發基於 Chrome/Edge 的網頁版控制台，取代傳統 ANSI 終端機。
    *   **即時狀態協定 (STX)**: 實作 Press/Release 雙向通訊，徹底解決終端機無法長按按鍵的限制。
    *   **全鍵盤模擬**: 支援完整 ASCII、CTRL 組合鍵與 F1-F3 系統功能。
    *   **智慧分流**: 實作選單模式自動判斷，方向鍵在 F3 選單下自動切換為導航模式。
*   **🌐 GitHub 部署**:
    *   Repo: `https://github.com/pondahai/PicoApple2`
    *   完成 Git 歷史安全清理（移除 ROM 檔）。
    *   專業級 README 與 GPIO 接線手冊。

## 🛠️ 硬體偏差補償 (Software Workarounds)
由於 PCB 接線失誤，目前程式碼中實作了以下對調（維持現狀）：
*   **S 鍵 <-> X 鍵**: Row 6/7 引腳對調。
*   **Page Up <-> '?'**: 搖桿按鈕 0 與鍵盤矩陣 '/' 對調。
